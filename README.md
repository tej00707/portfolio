# Tejovanth Kasula's Portfolio

A modern, responsive portfolio website built with React and Tailwind CSS to showcase projects, skills, and experience.

## 🚀 Features

- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI** - Built with Tailwind CSS for a professional look
- **Smooth Navigation** - Smooth scrolling between sections
- **Project Showcase** - Detailed information about projects with tech stack
- **Skills Display** - Organized skill categories with proficiency indicators
- **Contact Form** - Easy-to-use contact form for reaching out
- **Social Links** - Links to GitHub, LinkedIn, and email

## 📋 Sections

1. **Header** - Sticky navigation bar with smooth scroll links
2. **Hero** - Eye-catching introduction section
3. **About** - Personal background and education details
4. **Projects** - Showcase of key projects with descriptions and tech stack
5. **Skills** - Organized skill categories and coding profiles
6. **Contact** - Contact information and message form
7. **Footer** - Quick links and social media connections

## 🛠️ Tech Stack

- **React 18.2.0** - UI library
- **Tailwind CSS 3.3.0** - Utility-first CSS framework
- **React Icons 5.0.0** - Beautiful icon library
- **react-scripts 5.0.1** - Create React App scripts

## 📦 Installation

1. Navigate to the project directory:
```bash
cd portfolio
```

2. Install dependencies:
```bash
npm install
```

## 🚀 Running the Project

Start the development server:
```bash
npm start
```

The website will open in your browser at `http://localhost:3000`

## 🏗️ Building for Production

Create an optimized production build:
```bash
npm run build
```

The build folder will be ready to be deployed.

## 🎨 Customization

### Colors
Edit the color scheme in `tailwind.config.js`:
```javascript
colors: {
  primary: "#1f2937",
  secondary: "#3b82f6",
  accent: "#10b981",
}
```

### Content
Update your information in the respective component files:
- `About.jsx` - Personal information and education
- `Projects.jsx` - Project details and links
- `Skills.jsx` - Skills and coding profiles
- `Contact.jsx` - Contact information
